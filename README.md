# Pick-and-Place-Robot-using Quarky
This repository showcases a Pick and Place Robot built using the Quarky Robot. The project demonstrates basic automation where the robot picks objects from one position and places them at another using servo and motor control programmed in PictoBlox.

## üîå Getting Started with Quarky
1. Connect the battery (red wire facing right)
2. Secure battery using mounting parts and lock
3. Power ON Quarky using the switch
4. Follow the **pre-loaded firmware instructions**

---

## üíª Connecting Quarky with PictoBlox
1. Download and install **PictoBlox & PictoBlox Link**
2. Select **Quarky** as the board
3. Enable Bluetooth and connect
4. Green RGB light confirms successful connection

## üõ†Ô∏è Robot Assembly Summary
### Wheel Sub-Assembly
- Mount motors on the B1 plate
- Attach wheels
- Fix the assembly to the front of the robot
- Connect motors to Quarky motor ports

### Pick & Place Mechanism
- Mount servo motors on B5 plates
- Align the servo shaft with the ‚Äò+‚Äô mark
- Connect servos to **Servo Port 1 & 2**
- Ensure brown wire stays on the **left side**

### Servo Calibration
- Calibrate servos to **90¬∞**
- Ensures smooth movement and avoids motor strain

### Ultrasonic Sensor
- Mount using B2 & B7
- Connect Trig & Echo pins correctly
- Configure pins in code accordingly
